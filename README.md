Welcome to a super simple chess game developed using the MonoGame Framework!

This project is meant to be a showcase of object-oriented design and programming.


Executables can be found in Bundled/YourOperatingSystem

Example for windows: run the .exe file found in Bundled\win\SimpleChess.exe


Controls and Instructions for the Game:

 - The game is won upon capture of a king

 - Use your mouse to select and move pieces

 - Press the F key to toggle which side of the board you're viewing

 - Click anywhere to restart after a game has ended.
